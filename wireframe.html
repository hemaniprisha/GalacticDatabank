üõ†Ô∏è UIKit Implementation Notes

  • UITabBarController - Main container with 3 tabs (Home, Search, Favorites)
  • UINavigationController - Embed each tab's root view controller
  • UICollectionView - Category grid on Home screen (2 columns)
  • UITableView - List screens and search results
  • UISearchBar - Search functionality on Search screen and List screens
  • Custom UITableViewCell - Display thumbnail, title, and subtitle
  • UIScrollView - Detail screen with multiple sections
  • URLSession - Fetch data from Star Wars Databank API
  • UserDefaults - Store favorites locally
  • Codable - Parse JSON responses from API

üé® Design System

  • Primary Color: Gold (#FFD700) - Headers, active states, highlights
  • Background: Dark space theme (#1a1a2e, #2a2a3e)
  • Text: Light gray (#e0e0e0) for body, gold for titles
  • Borders: Subtle borders (#4a4a5e) for depth
  • Typography: System font (San Francisco), bold for titles
  • Icons: SF Symbols or emoji for categories

üìä API Endpoints Used

  • GET /api/v1/characters - List all characters (paginated)
  • GET /api/v1/vehicles - List all vehicles
  • GET /api/v1/species - List all species
  • GET /api/v1/locations - List all locations
  • GET /api/v1/droids - List all droids
  • GET /api/v1/creatures - List all creatures
  • GET /api/v1/organizations - List all organizations
  • GET /api/v1/characters/name/{name} - Get specific character

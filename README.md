# GalacticDatabank
A comprehensive Star Wars encyclopedia app that allows users to explore characters, vehicles, species, locations, droids, creatures, and organizations from the entire Star Wars universe using the official Star Wars Databank API. Users can search, browse by category, bookmark favorites, and discover detailed lore about any element of the saga.


## App Name: Galactic Databank 
### Tagline: "The Star Wars Universe in Your Pocket"
Description:
Galactic Databank is your comprehensive guide to the Star Wars universe. Explore detailed information about every character, vehicle, species, planet, droid, creature, and organization from the official Star Wars Databank. Whether you're watching your favorite saga, settling a debate with friends, or diving deep into lore, Galactic Databank gives you instant access to the knowledge of a galaxy far, far away.

## Required Features (MVP):
1.	Browse by Category - View paginated lists of all seven content types (Characters, Vehicles, Species, Locations, Droids, Creatures, Organizations)
2.	Search Functionality - Search across all categories to find specific entries
3.	Detail View - Display comprehensive information about any selected item including name, description, and image
4.	Favorites/Bookmarks - Save favorite entries for quick access later
### Optional Features (Stretch Goals):
1.	Advanced Filtering - Filter results within categories (by affiliation, era, etc.)
2.	Share Functionality - Share interesting facts with friends via social media
3.	Dark Mode Theme - Space-themed dark UI matching Star Wars aesthetic
4.	Random Discovery - "Surprise Me" button for random entry exploration
5.	Recently Viewed - History of browsed items for quick return access
6.	Offline Mode - Cache viewed content for offline access
